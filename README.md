## 🧩 대표 프로젝트
- **Healthcare 데이터 분석** — 공공보건/설문 데이터를 활용해 건강행태·만성질환 관리·지역 격차를 분석하고 근거 기반 인사이트를 도출
- **간호학생 교육용 인수인계 챗봇 설계·평가** — 임상 추론 기반 인수인계 훈련 시나리오 설계, 챗봇 대화 흐름/콘텐츠 개발, 평가 설계
- **만성질환자 모바일 앱 & Pill Box 개발·사용자 평가** — 복약 알림/기록/처방 촬영/데이터 연동 기능 중심의 앱과 디바이스 연계, 전문가/사용자 평가 수행

## 📌 진행 중/관심 작업
- 공공보건·의료 데이터를 활용한 **만성질환 관리 수준 및 지역 격차** 분석(가중치/복합표본 설계 반영)

- 만성질환자 대상 **복약 습관 형성** 기능 설계와 앱–Pill Box 연동 사용성/수용성 평가
